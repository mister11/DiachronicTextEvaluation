
EvaluationScript
================

To evaluate a DTE file use the following command in this directory:

perl DiachronicTaskEvaluation.pl goldStandard.txt machineOutput.txt textM


Usage::: perl DiachronicTaskEvaluation.pl file1 file2 granularity
Usage::: file1 is the goldStandard annotation name file
Usage::: file2 is the machine annotation name file
Usage::: granularity is a string , one and only one of these- textF, textM or textC - showing what granularity of the task is evaluated
